OS Nica character — placeholder folder.

Sprite sheets follow the same convention as admin/:
  base/<anim>-spritesheets/<variant>/osnica[-<weapon>]-<animKey><Direction>-spritesheet.png

Examples:
  base/idle-spritesheets/no-weapon/osnica-idleDown-spritesheet.png
  base/attack-spritesheets/sword/osnica-sword-attackUpLeftDownRight-spritesheet.png

Drop sheets in the matching anim subfolder; the Sprite Sandbox will pick
them up automatically once you select OS Nica from the character dropdown.
