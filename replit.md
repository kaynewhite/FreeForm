# FreeForm

A blank-slate Replit project. Currently serves a placeholder landing page so the
preview pane has something to display until you decide what to build.

## Stack
- Python 3.11 standard library (`http.server`)
- Static `index.html` placeholder

## Run
- Workflow `Start application` runs `python server.py` on `0.0.0.0:5000`
- The preview pane is proxied to this port

## Files
- `server.py` — minimal static file server with no-cache headers for dev
- `index.html` — placeholder landing page
